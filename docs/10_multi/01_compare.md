# collectionsタイプの比較


|   xxx   |     list      |    tuple    |     map     |     set     |
| :------ | :------------ | :---------- | :---------- | :---------- |
| 空宣言  | x = []        | x = ()      | -           | x = {}      |
| 空宣言2 | x = list()    | x = tuple() | x = map()   | x = set()   |
| 宣言2   | x = [1, 2, 3] | x = (1,2,3) | x = {1,2,3} | x = {1,2,3} |
| 追加    | x.append(4)   | -           | x['key1']=4 | x.add(4)    |
